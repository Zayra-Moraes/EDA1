# Ponteiros
- variaveis sao manipulaões na memoria em algum endereço. 
-  variáveis do tipo ponteiro podem ser 
utilizadas para manipular e armazenar endereços 
de memória
- Um ponteiro é uma variável que armazena um 
endereço de memória
- tipo_do_ponteiro *nome_do_ponteiro
````C
int i;   
int *p; // ponteiro 
````
- `&` pega o endereço de memoria; 
- Um ponteiro pode receber o endereço de uma 
variável utilizando o operador &
````C
int i;
int *p;
i = 0;
p = &i;// p recebe o endereço de memoria do i, 
print("%p", p)// %p imprimi o endereço em hexadecimal
````

- é possivel alterar o valor daquele endereço com o `*p`
    - olhar no slide 10 o exemplo
- E se você precisar inicializar o ponteiro com um 
valor “padrão”, antes de decidir qual o valor que 
ele irá receber? Para isso, utiliza-se a atribuição NULL

- ````C
    int *p = NULL;
    *p = 10; //pograma termina
  ````
    - nao tem como eu colocar algo em lugar nenhum, por isso antes de definir o valor eu tenho que definir o endereço na memória
- É possível utilizar os 
ponteiros dentro de 
condicionais, verificando 
se o seu conteúdo é 
válido ou comparando 
dois ponteiros

- ### Tipo void de ponteiro:
    - o tipo void pode receber variaveis de qualquer tipo, é generico, mas eu preciso fazer o casting (mudança) do tipo;  
    - Exemplo no slide 15

## Em funções :
- **passagem  por valor / passagem por referencia**
- Usa-se ponteiros em funções em C quando queremos mudar o valor de algum argumento(no escopo de cima)
- **exemplo no slide 29**
- usar ponteiros em função e mudar o argumento, -> **passagem por referencia**
- metodo convencional -> **passagem por valor**
