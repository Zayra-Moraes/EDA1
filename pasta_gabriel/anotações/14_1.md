# função recursiva
- uma função que se chama, como por exemplo uma função fatorial, chamada tambem uma função iterativa. 
- **caso base**, uma exceção, como 0! é igual a 1; tambem existe uma,  **hipotese indutiva/caso recursivo**
- entender mais sobre funções recursivas e suas capacidades de eficacias; 
- ordem:
    - receber um valor pra chamada
    - comparar entrada com o caso base
    - se for, retorno sera o valor definido para o caso
    - se nao, aplicar a hipotese de indução, abrindo uma nova chamada recursiva.
    - Esperar o retorno da nova chamada recursiva para definir o valor de retorno. 
    - **(Ver mais no slide )**
- cada versão da função vai criar suas variaveis separadas independentes; 
- O fato de cada versão guardar seus proprios dados , é chamado de **Estado**;
- as variaveis statics que sao compartilhadas entre as versões da mesma função.
