## Continuação Struct 
- Vetor de structs:
    - `variavel[num].campo = (...)`
    - funciona como um vetor normalmente de qualquer outro tipo de dado 
    - EX:
        ````C 
        struct aluno alunos[10], meu_aluno; // declarando um vetor e uma variavel aluno

        scanf("%s", alunos[0].nome);
        scanf("%f", &meu_aluno.notas[0]);

        ````

- structs podem ser associados, mas é recomendado que os valores sejam associados separadamente; 

obs: `sizeof(x)` -> me retorna o tamanho da vairavel em bites 

- ## TypeDef

    ````C
    typedef int inteiro;

    int i;   // são a mesma coisa

    ````
    -  normalmente sao usados com structs pra deixar o codigo mais simples 
    ````C
    typedef struct campos_de_um_ret{

    int main(){
        retangulo ret;// funcional e representa o struct retalgulo
    ````

    - convenção de sintaxe `_retangulo` -> `struct`
