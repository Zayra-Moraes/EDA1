# STRINGS
- posições adjascentes no vetor tbm estao alocadas de modo igual(adjascente) na memoria do computador ;
- diferença entre String e um vetor de char, a string termina com o '\0' que representa um final de uma string, detalhe, ele ocupa um espaço de caracter; 
- posso inicializar :
```C
    char str[10] = "oi"; //-> aqui ja vem com o '\0 inbutido'
    char str[10] = {'o','i', '\0'};
```
- lembrando que a variavel considerada String nao pode ser redeclarada com um valor diferente
- nao pode inicializar ela e depois declar, nem associar uma a outra
- elas podem ser manipuladas como Vetores normalmente 
```C 
    char str[10] = "ola";
    str[1] = 'p'; // virou 'opa'
````

# Struct 
- serve pra agurpar tipos diferentes de dados dentro de uma variavel; 
- declaração: 
````C 
    struct minha_struct{
        tipoA campoA;
        tipoB campoB;
    };

    struct aluno{
        char nome[100];
        int matricula;
    } aluno1,aluno2;
````
- alem disso é possivel declarar variaveis junto que no caso é `aluno1, aluno2`.
- ela fica entre a `main()` e os imports/includs,fica no comecinho do arquivo. 
- como utilizar as variaveis de dentro do struct ? 

````C
 int main(){
    struct aluno joao;
    int i;
 

    scanf("%s", joao.nome);
    scanf("%d", &joao.matricula); // joao.nome, -> acesso as  variaveis do meu struct
    return 0; 
}

// posso fazer como um objeto tbm 
int main(){
    struct aluno = {"nome do aluno", 242004706}
}


// jeito moderno
int main(){
    struct aluno gabriel={
        .nome = "Gabriel",
        .matricula = 242004706,
    };// aqui nao precisa ser na mesma ordem da declaração 
}
````
- nao tem com eu colocar nenhum tipo de função dentro do struct
